    参数必写

    npm run deployHtml --b 'test' --cm 'autp deploy for new tool 3'

    release 部署
    npm run deployHtml --b 'release' --cm 'autp deploy for new tool 3'
    npm run deployStatic --b 'release' --cm 'this is common s for autp deploy for new tool 3'
