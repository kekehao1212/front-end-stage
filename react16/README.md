## 介绍

  技术栈：React + React-Route + Redux + Antd
  功能模块：
    1. 支持开发热更新
    2. 提供本地mock服务
    3. 提供proxy代理服务
    4. 可生产模式预览
    5. 引入antd，按需加载，本地化字体和主题定制

## 开发启动
 
  npm i
  npm run start

 ## 打包

  npm run build

## 生产预览

  npm run preview
